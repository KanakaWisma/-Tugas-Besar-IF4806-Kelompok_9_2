# -Tugas-Besar-IF4806-Kelompok_9_2
Multi-linked list yang memodelkan poli klinik, pasien, dan hubungan di antara mereka. Satu poli dapar diisi oleh banyak dokter serta satu dokter dapat menangani banyak pasien. Pasien dapat mengunjungi setiap poli sesuai jadwalnya. Program ini ditujukan untuk mencatatan setiap kunjungan yang ada di poli tersebut.
